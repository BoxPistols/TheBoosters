# The Boosters — UI コンセプト「クラシックロック」

Status: Proposal (2026-07-05) / Owner: BoxPistols
関連: docs/NEXT-STEPS-2026-06.md, .claude/skills/boostnote-modernize

## ねらい

「The Boosters」はロカビリーバンドのような名前を持つ。上流 Boostnote の痕跡を
消した今、視覚アイデンティティも独自に立てる。方向性は **ビートルズ〜ロカビリー
のクラシカルなロックテイスト**：ヴィンテージだが古臭くない、ステージの熱と
道具（楽器・アンプ）の質実さを持つ UI。

## 方向性 3 案（推奨: A）

### A. Vintage Stage（推奨）

チューブアンプとステージ照明の世界観。ダークベース維持で移行コスト最小、
長時間のコーディング/執筆にも目に優しい。

| Token | 値 (案) | 用途 |
|---|---|---|
| `bg-stage` | `#1C1A17`（ウォームチャコール） | アプリ背景 |
| `bg-panel` | `#26231F` | サイドバー/リスト |
| `cream` | `#F2E8D5`（ヴィンテージクリーム） | 本文テキスト |
| `accent-red` | `#D7263D`（バーミリオン） | 選択・アクティブ（現ピンクの後継） |
| `amber` | `#E8A33D`（真空管アンバー） | ホバー・警告・スター |
| `chrome` | `#8C949E` | 罫線・アイコン非活性 |

- 見出し/ロゴだけレトロ Slab 系（例: Alfa Slab One 相当）を点で使う。本文・
  コードは現行のまま（可読性最優先）
- モチーフ: ローディング＝回転するビニールレコード、タグ＝チケット半券、
  進捗/文字数＝VU メーター
- 現在のピンク `#E10051` 系アクセントを `accent-red` に置換するだけでも
  第一印象が変わる

### B. Sunburst Guitar

レスポール/ストラトのサンバースト塗装（アンバー→ブラウンのグラデ）＋
べっ甲柄アクセント。温かみは最強だが、グラデ多用は実装・視認性コストが高い。
A のアクセント拡張（イラスト・空状態画面）として部分採用が現実的。

### C. Mod / '60s Beatles

白黒ベース＋幾何学的ポップアクセント（ライトテーマ向き）。クリーンだが
「ノートアプリの没個性」に近づくリスク。ライトテーマ刷新時の候補として保留。

## 実装フェーズ（A 採用時）

既存テーマ機構（`browser/styles/index.styl` の `$themes = 'monokai' 'nord'
'vulcan'` + `theme/dark.styl`、config `ui.theme`）にそのまま載せる。

1. **Phase 1 — カラーテーマ 'rockabilly'**（~0.5 日）
   `$themes` に追加し上表のトークンで配色。Preferences > Interface から選択可。
   既存テーマは壊さない（追加のみ）。
2. **Phase 2 — ブランドアセット**
   アプリアイコン刷新（現 `resources/app.png` は Boostnote の緑ロゴ = 痕跡）。
   案: ギターピック型に「B」/ビニールレコード。トレイ・Dock・About・
   electron-builder の icon 一式を差し替え。
3. **Phase 3 — モチーフ & マイクロインタラクション**
   ローディング・空状態・スプラッシュにモチーフ導入。
   `prefers-reduced-motion` 尊重。
